# VinWin FAQ

**Сайт:** https://artyompetrenko.github.io/vinwin-faq/

Статический сайт с ответами на базовые вопросы про нейронные сети, LLM и AI.

**Цель:** перестать отвечать на одни и те же вопросы вручную — давать ссылку.

---

## Для кого

Для людей, которые вообще не знают, что такое нейронная сеть. Никакого жаргона, простые объяснения.

## Что внутри

- **Что такое** — базовые понятия: нейронная сеть, LLM, токен, контекст и др.
- **Ограничения и мифы** — чего нейронки не умеют и что про них понимают неправильно

## Как устроен сайт

Два режима отображения, переключаются кнопкой:
- **Одна страница** — все вопросы аккордеоном
- **По категориям** — навигация по разделам

Каждый вопрос имеет якорную ссылку для шаринга.
