### Этап 2 — HTML-структура
**Критерий готовности:** вся разметка есть, контент-заглушки на месте, якоря работают.

Структура `index.html`:

```
<header>
  Название + кнопка переключения режима

<nav>
  Ссылки на категории (#chto-takoe, #mify)

<main>
  <section id="chto-takoe">
    <h2>Что такое</h2>
    <div class="qa-item" id="chto-takoe-llm">
      <button class="question">...</button>
      <div class="answer">...</div>
    </div>
    ...

  <section id="mify">
    <h2>Ограничения и мифы</h2>
    ...
```

- [ ] Header с заголовком и кнопкой-переключателем
- [ ] Nav с якорями на категории
- [ ] Две секции с заглушками (3-5 вопросов-заглушек в каждой)
- [ ] Уникальный `id` у каждого вопроса (формат: `slug-вопроса`)

### Этап 3 — CSS / Дизайн
**Критерий готовности:** сайт выглядит законченно, читаемо, дружелюбно для новичка.

Токены дизайна:
- Шрифт: системный стек (`-apple-system, Segoe UI, sans-serif`)
- Фон: белый `#ffffff`, акцент: мягкий синий `#4A90D9`
- Текст: тёмно-серый `#1a1a1a`, вторичный `#555`
- Максимальная ширина контента: `760px`, центрирован
- Отступы: `1rem` / `1.5rem`

- [ ] Reset + base typography
- [ ] Header: название слева, кнопка-переключатель справа
- [ ] Nav: горизонтальные ссылки-таблетки
- [ ] Аккордеон: вопрос — кликабельная строка со стрелкой; ответ — блок под ним
- [ ] Стили для двух режимов (классы `mode-single` / `mode-multi` на `<body>`)
- [ ] Кнопка-переключатель: чёткий toggle с подписью текущего режима
- [ ] Базовый responsive: не ломается при ширине 375px

---

### Этап 4 — JavaScript
**Критерий готовности:** оба режима работают, позиция сохраняется, якоря работают.

#### Аккордеон
- Клик по вопросу → `aria-expanded` toggle, высота ответа анимируется
- Только один открытый вопрос в категории (или все сразу — решить при реализации)

#### Переключатель режимов
- Режимы: `single` (по умолчанию) и `multi`
- Класс на `<body>` управляет всей разницей в отображении через CSS
- Сохранение позиции при переключении:
  - `single → multi`: определить, какая секция сейчас видна в viewport → показать её в multi
  - `multi → single`: запомнить активную категорию → прокрутить к ней в single

#### Якоря
- При загрузке страницы с `#id`: открыть нужный вопрос и прокрутить к нему
- Кнопка «скопировать ссылку» рядом с каждым вопросом (опционально, после MVP)

- [ ] Аккордеон
- [ ] Переключатель режимов
- [ ] Сохранение позиции при переключении
- [ ] Обработка якоря при загрузке

**Критерий готовности:** заглушки заменены реальными вопросами и ответами.

Категория **«Что такое»** — примерный список:
- Что такое нейронная сеть?

- Нейронка запоминает наши разговоры?
