## Публикация

### 1. Создать репозиторий на GitHub

Через браузер или `gh`:

```bash
gh repo create vinwin-faq --public --source=. --remote=origin
```

Или вручную: создать репо на github.com, скопировать remote URL, затем:

```bash
git remote add origin https://github.com/<username>/vinwin-faq.git
```

### 2. Финальный коммит и push

```bash
git add -A
git commit -m "feat: complete FAQ site ready for launch"
git push -u origin main
```

### 3. Включить GitHub Pages

1. Открыть репозиторий на github.com
2. Settings → Pages
3. Source: **Deploy from a branch**
4. Branch: `main`, Folder: `/ (root)`
5. Save

Сайт появится через ~1 минуту по адресу:  
`https://<username>.github.io/vinwin-faq/`

### 4. Проверить по публичному URL

- [ ] Сайт открывается
- [ ] CSS загружен (не голый HTML)
- [ ] JS работает (аккордеон, переключатель)
- [ ] Якорные ссылки работают на публичном URL
